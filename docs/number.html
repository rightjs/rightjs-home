Number

RightJS extends the Number unit in order to make it support the Math module methods in a more natural, object-oriented way. And it adds the handy Ruby style loop methods as well.

Methods

include, abs, ceil, downto, floor, max, min, round, times, to, upto

top ↑include

Number.include(Object new_methods[, Boolean dont_overwrite) -> Number

Description

Registers new functionality for Number instances

If the second argument is true, then the method will skip already existing methods

Example

Number.include({
double: function() {
return this * 2;
}
});

2..double(); // -> 4

top ↑abs

abs() -> number absolute value

Description

Returns the absolute value of the number.

Example

-1..abs(); // -> 1
22..abs(); // -> 22

top ↑ceil

ceil() -> number

Description

Returns the smallest integer bigger than or equal to the current number.

Example

2.2.ceil(); // -> 3

top ↑downto

downto(number end, Function lambda[, Object scope]) -> number self

Description

Iterates the given lambda function from the current number down to the given stop number:

Example

var numbers = [];

8..downto(4, function(i) {
numbers.push(i);
});

numbers; // [8,7,6,5,4]

top ↑floor

floor() -> number

Description

Returns the biggest integer smaller than or equal to the current number.

Example

2.2.floor(); // -> 2

top ↑max

max(number limit) -> number

Description

Limits current number but the given one

Example

4..min(6); // -> 4
8..min(6); // -> 6

top ↑min

min(number limit) -> number

Description

Limits current number but the given one

Example

4..min(2); // -> 4
1..min(2); // -> 2

top ↑round

round([Integer base]) -> number

Description

Returns the integer number closest to the current number. The optional argument specifies how many digits after the decimal point should be retained.

Example

2.2.round(); // -> 2
8.8.round(); // -> 8

4.444.round(1); // -> 4.4
4.444.round(2); // -> 4.44
4.444.round(3); // -> 4.444

top ↑times

times(Function lambda[, Object scope]) -> number self

Description

Call the given lambda function as many times times as specified by the current number:

Example

var numbers = [];
4..times(function(i) {
numbers.push(i);
});

numbers; // -> [0,1,2,3]

top ↑to

to(number end, Function lambda[, Object scope]) -> number self

Description

Same as upto, downto but automatically decides which direction to count

Example

1..to(4, function() {}); // -> counts from 1 to 4
4..to(1, function() {}); // -> counts from 4 to 1

top ↑upto

upto(number stop, Function lambda[, Object scope]) -> number self

Description

Iterates the given lambda function from the current number up to the given stop number:

Example

var numbers = [];

4..upto(8, function(i) {
numbers.push(i);
});

numbers; // -> [4,5,6,7,8]
